Some example for running:

* find suspicious activities:
    http://localhost:8080/api/suspicious?startTime=2020-02-22%2016:50:39&endTime=2020-08-22%2016:50:39

* see the statistic for specific game run:
    http://localhost:8080/api/game-statistics?startTime=2010-02-22%2016:50:39&endTime=2050-04-22%2016:50:39&game=pocker

* see the statistics for all games, run:
    http://localhost:8080/api/game-statistics?startTime=2010-02-22%2016:50:39&endTime=2050-04-22%2016:50:39&game
